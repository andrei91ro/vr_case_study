Adding 3D models

Creating a Graphical User Interface

Graphical User Interfaces (GUIs) are available in Virtual Reality. This example showcases the use of the Gaze cursor that allows the user to select items.

In this example, the user must first gaze at a product and afterwards at a specific button, once the product was selected.

Enhancing products

A-Frame opens endless posibilities of enhanging user experience. This example shows how one can project a movie onto a texture such that it can be rendered inplace of a black TV screen.
